System requirements


  • Java Runtime Enviroment Version 1.6 or higher
  • TimeNet Version 4.02 or higher
    • on Linux and other X-Systems gcc and autotools, needed for local simulation
    • on Windows MinGW and MSys (install base packages with installer)
  • R (Version 3.1 is recommended)
    • Plot3D-package
    • R-OpenGL-package
